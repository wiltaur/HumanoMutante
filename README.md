# HumanoMutante

- Este proyecto fue generado usando Angular CLI version 19.2.27.
- Se utiliza Bootstrap para los estilos.
- El desarrollo está basado en Listas de string y matrices que representan la tabla de las bases nitrogenadas (A, T, C, G).
- Son 3 pasos para su uso:
  1. Elegir el tamaño desde (4x4)
  2. Llenar la tabla dando clic en cada celda donde cada clic cambia entre los valores permitidos (A, T, C, G)
  3. Dar clic en el botón "Analizar ADN", devolverá si la persona es o no mutante y se verán en verde las celdas que correspondan

## Compilación en Desarrollo

Para compilar en local, ejecute:

```bash
ng serve -o
```
